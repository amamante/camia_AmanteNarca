# TaskMaster

**A Web-Based Study Planner for Academic Task Management**

## Problem Statement

Students often struggle with managing multiple academic tasks, deadlines, and subjects efficiently. Traditional methods like notebooks or scattered notes make it difficult to track progress, prioritize work, and stay organized. There is a need for a simple, centralized, and user-friendly system that allows students to manage their academic responsibilities effectively.

## Project Description

TaskMaster is a web-based task management system designed to help students organize and manage academic responsibilities efficiently.

It allows users to create, track, and manage tasks such as assignments, projects, and deadlines in a clean and structured interface. The system includes features like subject organization, priority levels, countdown timers, and a calendar view to improve productivity and time management.


## Project Objectives

* Organize academic tasks by subject
* Allow users to create, edit, and delete tasks easily
* Track task completion status (pending vs completed)
* Manage deadlines with date and time inputs
* Provide a visual calendar for better planning
* Improve time management through countdown tracking
* Ensure accessibility and ease of use through a clean UI


## Planned Features

* Subject Management System
* Task Management (CRUD functionality)
* Task Priority Levels (High, Medium, Low)
* Deadline Tracking (Date & Time)
* Live Countdown Timer
* Calendar View of Tasks
* Task Completion System
* Local Storage (data persists after refresh)
* Notification/Toast Message on task completion


## How to Run the Program

1. Download or copy the project file
2. Open the .html file in any browser (Chrome recommended)
3. No installation required
4. Start using TaskMaster immediately
